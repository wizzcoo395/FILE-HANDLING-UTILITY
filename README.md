# File Handling Utility (Java)

## About This Project

This is a simple Java program I created to understand how file handling works in real life. Instead of just learning theory, I wanted to actually try writing, reading, and updating data inside a file using Java.

The program works on a basic menu system where the user can choose what they want to do. It uses a text file (`sample.txt`) to store all the data. This project helped me get a clear idea of how Java interacts with files and how data can be saved permanently.

---

## What This Program Does

### 1. Write Data

The user can enter any text, and it gets saved into the file.
If the file already has content, it will be replaced with the new data.

### 2. Read Data

This option displays whatever is stored in the file.
It reads line by line and prints it on the screen.

### 3. Modify (Append)

Instead of deleting old data, this option adds new content at the end of the file.
This is useful when we want to keep previous records.

## How to Run

1. Open terminal / command prompt
2. Go to the folder where the file is saved

Compile:
javac FileHandlingUtility.java

Run:
java FileHandlingUtility

## File Structure

## Example Run

When you start the program, you’ll see options like:

1. Write to File
2. Read from File
3. Modify File
4. Exit

You just need to enter the number and follow the instructions.
